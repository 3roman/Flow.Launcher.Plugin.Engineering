这是一个[Wox](https://github.com/Wox-launcher/Wox)插件项目，使用 C# 开发，目前完成了以下插件：

- Wox.Plugin.SteamTable

  根据压力、温度查询水蒸气的性质

- Wox.Plugin.GoogleTranslate

  借助谷歌翻译 API 互翻中英文

- Wox.Plugin.PressureRatingConverter

  互转 Class 与 PN 压力等级

- Wox.Plugin.PipeOutsideDiameter

  根据管道公称直径查询公英制外径

- Wox.Plugin.Pipe

  根据管道公称直径和管表号查询外径和壁厚
